# __REPO_NAME__

Production-ready Next.js App Router starter scaffolded by `dev-toolbox`.

## Included by default

- Next.js App Router + TypeScript
- Tailwind CSS baseline
- shadcn-style UI foundation (`components/ui` + `cn` helper)
- Radix UI primitives
- Vercel Analytics + Speed Insights

## Quickstart

```bash
npm install
npm run dev
```

## Core scripts

- `npm run dev`
- `npm run lint`
- `npm run typecheck`
- `npm test`
- `npm run build`

## Optional scaffold extras

When scaffolding from `dev-toolbox`, you can enable:

- `--motion` for Framer Motion starter components/routes
- `--remotion` for video composition scaffolding
- `--vercel-agents` for Vercel AI SDK agent route + demo page
- `--storybook` for Storybook with a11y/docs/links/themes/onboarding addons
- `--claude` for a `CLAUDE.md` collaboration scaffold
