# Code of Conduct

## Our commitment

We want this project to be a professional, respectful place for everyone.

## Expected behavior

- Be constructive and specific.
- Focus critique on code and behavior, not people.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions tied to project outcomes.

## Unacceptable behavior

- Harassment, discrimination, threats, or personal attacks.
- Trolling, derailing, or intentionally inflammatory comments.
- Publishing private or sensitive information without consent.

## Enforcement

Maintainers may remove comments, close threads, or block contributors when necessary to protect the project.

## Reporting

Report conduct issues privately to:

- mail@digitaltableteur.com

Include links/screenshots and relevant context.
